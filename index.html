Sección 1

¿Que es la Sociedad Digital?

La sociedad digital se refiere a la estructura social y cultural influenciada por las tecnologías de la información y comunicación (TIC). En este entorno, la interacción, producción, consumo y difusión de información ocurren a través de plataformas digitales, como internet, redes sociales, inteligencia artificial y la economía digital. Esto ha transformado la organización y participación en la sociedad. Un ejemplo es cómo la digitalización ha permitido enviar correos instantáneamente en lugar de esperar meses para su entrega.

Los Datos

En la sociedad digital, los datos son información sin procesar que se puede almacenar, transmitir y manipular por medio de sistemas computacionales.

Con base a estos datos se pueden idear y realizar distintas actividades y tareas útiles tanto para el ámbito personal como para el corporativo, dentro de las cuales están:

  1. Código ASCII
  2. Ondas Análogas y Digitales
  3. Código Binario
  4. Pirámide DIKW

  1. Código ASCII

    El código ASCII (American Standard Code for Information Interchange) es un sistema de codificación que asigna un número a cada letra, número, símbolo o carácter de control, permitiendo que las computadoras representen texto. Se usa para facilitar la comunicación entre dispositivos electrónicos. ASCII incluye 128 caracteres, como letras, números y símbolos comunes.

    Ejemplos:

    • A=65
    • a=97
    • 0=48
    • Espacio=32

    El código ASCII es fundamental para la representación y manipulación de texto en sistemas informáticos

  2. Ondas Análogas y Digitales

    Las ondas análogas son señales continuas que varían en el tiempo, como las usadas para transmitir sonido o luz. El código ASCII, por otro lado, es un sistema de codificación digital que representa caracteres. Aunque no están directamente relacionados, las ondas análogas se digitalizan para ser procesadas, y ASCII ayuda a representar texto en ese formato digital.

  3. Código Binario

    El código binario es un sistema de representación de datos usando solo dos dígitos: 0 y 1. Es la base del funcionamiento de las computadoras, ya que todo lo que procesan se convierte en secuencias binarias. El código ASCII usa binario para representar letras, números y símbolos. Por ejemplo, la letra "A" en ASCII (65) se convierte en 01000001 en binario. Las señales análogas (como el sonido) se digitalizan en binario para ser procesadas por dispositivos electrónicos. Así, todo se transforma en ceros y unos para ser manejado por la tecnología digital.

  4. Pirámide DIKW

    5. La Pirámide DIKW es un modelo que respresenta la jerarquía del conocimiento, se compone por cuatro niveles:

    1. Datos: Hechos sin procesar, como números y/o símbolos.
    2. Información: Los datos procesados y organizados para tener significado.
    3. Conocimiento: La comprensión y análisis de la información para tomar decisiones.
    4. Sabiduría: La capacidad de aplicar el conocimiento con juicio para resolver problemas y prever consecuencias.

Sección 2

Hardware

El Hardware es el conjunto de componentes físicos de una computadora, los cuales permiten que el sistema funcione y ejecute las tareas que se indiquen. Se divide en diferentes categorias:

  1. Dispositivos de entrada: Permiten al usuario interacturar con el equipo o computadora.
  2. Dispositivos de Salida: MUestran la información al usuario, como el monitor y la impresora.
  3. CPU (Unidad Central de Procesamiento): Es el cerebro del sistema, este se encarga de ejecutar las instrucciones de los programas.
  4. Memoria: Se usa para almacenar datos, así de forma temporal (RAM) como pemanente (disco duro, SSD)
  5. Placa Base: Conecta todos los componenetes y permite su comunicación.
  6. Tarjetas de Expansión: Mejoran funciones específicas, como son los gráficos o el sonido.
  7. Fuente de Alimentación: Proporciona energía eléctrica a todos los componentes.

Software

El Software es el conjunto de programas y aplicaciones que permiten que una computadora o dispositivo realice tareas específicas. Este es intangible y se compone por líneas de código que le indican al hardware que hacer. El Software se divide en varios tipos:

  1. Software de Sistema: Gestiona el hardware y las operaciones del sistema, como los sistemas operativos como lo son Windows o macOS y también gestiona los controladores de dispositivos.
  2. Software de aplicación: Son programas que realizan tareas específicas, como procesadores de texto como MS Word, navegadores web como Chrome y/o aplicaciones Móviles como X.
  3. Software de Programación: Herramientas que los programadores usan para crear sus programas, como lenguajes de programación como son JavaScript o Python.

Sección 3

Redes

Una red es un conjunto de dispositivos interconectados que intercambian informacion, se conectan mediante componentes físicos dentro de la misma, existen más de 7 tipos de red, de los cuales, los 3 principales son:

  1. Red LAN: Las redes tipo LAN (local area network) son todas aquellas redes que abarcan un área geográfica pequeña, generalmente estan echas para hogares y redes con una capacidad maxima de 3 conectados por conexión cableada
  2. Red WLAN: La red WLAN (wireless local area network) es una red parecida a la LAN, pero esta es inalambrica y con un area geográfica un poco mayor.
  3. Red WAN: La red WAN (wide area network) es la red con el área geográfica más grande, esta generalmente es usada en empresas e instituciones bastante grande, el ejemplo más cercano seria el de la institucion escolar APDE EL ROBLE, que usa este tipo de red.

Componentes de una red (LAN)

  1. Servidor: Es un aparato informático que almacena, distribuye y suministra información.
  2. Switch: Es el dispositivo de interconexión utilizado para conectar equipos en red formando lo que se conoce como una red de área local (LAN)
  3. Routher: Es un dispositivo que transimite señal de internet (puede ser de forma inalambrica) hacia los clientes
  4. Clientes: es la entidad de software que realiza las peticiones de servicio, es decir, lo que usa todos los servicios de internet y trabaja con ellos.
  5. Internet: este se considera un componente de la red, al ser necesario conectarlo al switch y al routher, ademas de que sin este la red queda limitada a las tareas fuera de línea del usuario.
  6. Punto de acceso(AP): este componente se usa solamente en redes de tipo WAN, y su funcion es ampliar la señal de internet proporcionada por el rauther, para conectar más dispositvos sin tener que tener más de una red contratada

Conceptos del Internet

  1. ¿Que es el Internet?: es la entidad de software que realiza las peticiones de servicio.
  2. Dominios: Es la entidad de software que realiza las peticiones de servicio.
  3. ¿Como funciona?: El internet funciona con muchos componentes difrerentes, los cuales son:
    • Servidores: Son computadoras que almacenan y gestionan recursos como sitios web, aplicaciones y datos.
    • Clientes: Son dispositivos (como computadoras o teléfonos) Que solicitan y reciben informacion a los servidores
    • ISP (Proveedor de servicios de internet): Es la empresa que proporsiona internet a los usuarios.
    • Protocolos: Son los conjuntos de reglas que permiten la comunicación entre dispositivos en la red
    • DNS: (sistemas de nombres de dominio): Es el servidor que traduce nombres de dominio amigables (como www.ejemplo.com) a driecciones IP que las computadoras pueden entender.
  4. Seguridad y privacidad: Hoy en día el internet es cada vez más y más grande, por lo que también se vuelve más inseguro, por lo que es importante tener en cuenta la seguridad personal, protegiendo tus cuentas con herramientas como contraseñas, cuentas privadas y muchas otras cosas por el estilo.

Sección 4

Introducción a la inteligencia artificial

La inteligencia artificial es un campo de la ciencia relacionado con la creación de computadoras y máquinas que pueden razonar, aprender y actuar de una manera que normalmente requeriría inteligencia humana o que involucra datos cuya escala excede lo que los humanos pueden analizar.

Basicamente la IA (Inteligencia Artificial) es un programa con la capacidad de pensar, para que este tenga multiples funciones de asistencia y conocimiento

Comienzos de la IA

La historia de la inteligencia artificial (IA) se remonta a la antigüedad, cuando se contaban mitos y leyendas sobre seres artificiales que poseían inteligencia o conciencia, creados por hábiles artesanos. Los fundamentos de la IA moderna fueron establecidos por filósofos que intentaron explicar el pensamiento humano como una manipulación mecánica de símbolos.

Este esfuerzo culminó en la invención de la computadora digital programable en los años 40, una máquina que se basa en los principios abstractos del razonamiento matemático. Esta invención y las ideas que la rodeaban motivaron a algunos científicos a comenzar a explorar seriamente la posibilidad de crear un cerebro electrónico.

Aplicaciones de la IA

Como todo avance tecnológico, la inteligencia artificial tiene sus aplicaciones en la vida cotidiana de las personas hoy en día, algunas de estas son.

  • Atención al Cliente: Un buen ejemplo es Zara, la famosa cadena de moda. Zara utiliza inteligencia artificial en su atención al cliente a través de chatbots en su sitio web y aplicaciones móviles. Estos chatbots pueden responder preguntas frecuentes, ayudar a los clientes a encontrar productos y gestionar devoluciones. Además, analizan las interacciones para mejorar continuamente la experiencia del usuario, personalizando recomendaciones de productos y haciendo que la atención sea más eficiente. (fuente: the verge; periodico en línea)
  • Diagnóstico Médico: Un buen ejemplo es Google Health, que ha desarrollado modelos de IA para detectar enfermedades oculares a partir de imágenes de retina, logrando diagnósticos con alta precisión.(fuente: The verge; periodico en línea)
  • Reconocimiento de Voz: Un ejemplo destacado es Amazon, con su asistente virtual Alexa. Alexa utiliza inteligencia artificial para el reconocimiento de voz, permitiendo a los usuarios interactuar con dispositivos inteligentes, hacer preguntas, reproducir música y controlar el hogar a través de comandos de voz.(fuente: TechCruss; sitio web informativo)
  • Predicción de Tendencias de Mercado: Un ejemplo destacado es Bloomberg, que utiliza inteligencia artificial en su plataforma de análisis de datos financieros. Bloomberg Terminal emplea algoritmos de IA para analizar grandes volúmenes de datos del mercado y generar predicciones sobre tendencias, lo que ayuda a los inversores a tomar decisiones informadas.(fuente: Business Insider; sitio web informativo)
  • Transporte Autónomo: Un buen ejemplo es Tesla, que ha implementado su sistema de conducción autónoma, conocido como Autopilot, en sus vehículos. Utiliza IA para ayudar en la navegación, el cambio de carriles y otras funciones de conducción.(fuente: Electreck; pagina web informativa)
  • Agricultura de Precisión: Un excelente ejemplo es Monsanto, que ahora forma parte de Bayer. Esta empresa utiliza inteligencia artificial y tecnología de datos para mejorar la agricultura de precisión. Su plataforma, Climate FieldView, integra datos climáticos, de suelo y de cultivos para ayudar a los agricultores a tomar decisiones informadas sobre el uso de insumos, riego y cosechas.(Fuente: Bayer.com; periodico en línea)

Ramas de la IA

Como todos los conceptos electronicos, la IA tiene sus ramas de funcionamiento, algunas de ellas son:

  • Aprendizaje Automático: El aprendizaje automático (machine learning) es una rama de la inteligencia artificial que permite a las máquinas aprender de datos y mejorar su rendimiento en tareas específicas sin ser programadas explícitamente para cada tarea. Utiliza algoritmos para identificar patrones en los datos y hacer predicciones o tomar decisiones basadas en esos patrones. Un ejemplo de esto pude ser un sistema de recomendación de películas en una plataforma de streaming. Este sistema utiliza aprendizaje automático para analizar los hábitos de visualización de los usuarios.
  • Procesamiento de Lenguaje Natural (PLN): Es un subcampo de la inteligencia artificial que se centra en la interacción entre las computadoras y el lenguaje humano. Su objetivo es permitir que las máquinas comprendan, interpreten y generen lenguaje humano de manera que sea valiosa y útil. Esto incluye tareas como traducción automática, análisis de sentimientos, chatbots, y más. Un ejemplo de esto puede ser un asistente virtual, como Siri o Alexa, utiliza PLN para entender y responder a las preguntas de los usuarios. Por ejemplo, si un usuario dice: "¿Cuál es el clima hoy?", el asistente procesa esa pregunta, identifica la intención del usuario (querer saber el clima) y busca la información relevante para responder de manera apropiada, como "Hoy hará 22 grados y estará soleado."
  • Visión por computadora: La visión por computadora es una rama de la inteligencia artificial que permite a las máquinas interpretar y entender el contenido visual del mundo, utilizando imágenes y videos. Esto implica tareas como la identificación de objetos, la detección de rostros, la segmentación de imágenes y el reconocimiento de patrones. Un ejemplo común es el uso de la visión por computadora en los coches autónomos. Estos vehículos utilizan cámaras y sensores para "ver" su entorno. A través de algoritmos de visión por computadora, pueden identificar peatones, otros vehículos, señales de tráfico y obstáculos, lo que les permite tomar decisiones en tiempo real para navegar de manera segura.

Tipos de algoritmos de aprendizaje

Existen 3 tipos principales de algoritmos de aprendizaje en el mundo de la IA, estos son:

  1. Aprendizaje supervisado
  2. Aprendizaje no supervisado
  3. Aprendizaje reforzado

  • Aprendizaje Supervisado

    • El aprendizaje supervisado es una forma de enseñar a una computadora a hacer predicciones. Se le muestra un conjunto de ejemplos donde cada ejemplo tiene una respuesta correcta (como fotos de perros y gatos con sus etiquetas correspondientes). La computadora aprende de estos ejemplos para poder adivinar la respuesta correcta cuando le muestras algo nuevo. Es como aprender con un maestro que te dice si estás en lo correcto o no.

  • Aprendizaje no supervisado

    • El aprendizaje no supervisado es una forma de enseñar a una computadora a encontrar patrones en datos sin tener respuestas correctas. En lugar de mostrarle ejemplos con etiquetas, le das un conjunto de datos y le permites descubrir por sí misma las relaciones y estructuras que hay en ellos. Por ejemplo, si le das a la computadora muchas fotos sin etiquetas, podría agruparlas en diferentes categorías basándose en similitudes, como agrupar fotos de animales o paisajes. Es como explorar algo nuevo sin saber de antemano qué buscar.

  • Aprendizaje reforzado

    • El aprendizaje reforzado es una forma de enseñar a una computadora a tomar decisiones mediante prueba y error. En este enfoque, la computadora aprende a través de recompensas y castigos. Imagina que estás entrenando a un perro: le das una galleta (recompensa) cuando hace algo bien, y no le das nada (castigo) cuando no lo hace. De manera similar, en el aprendizaje reforzado, el modelo realiza acciones en un entorno y recibe retroalimentación. Con el tiempo, aprende a elegir las acciones que le traen más recompensas.

Aspectos Éticos de la IA

En todo el desarrollo de la IA, aun hay mucho estudio por hacer, ya que aun hay muchas dificultades en este campo, algunos de estos son:

  • Sesgo en los algoritmos

    • Cuando un algoritmo se entrena con datos que reflejan prejuicios o desigualdades existentes, puede aprender esos mismos patrones. Por ejemplo, si un modelo se alimenta principalmente de ejemplos de un grupo específico y no de otros, puede llegar a tomar decisiones que favorecen a ese grupo y marginan a los demás. Además, el diseño del propio algoritmo puede influir en estos sesgos, ya que algunas características pueden recibir más peso que otras. Esto es problemático, especialmente en contextos como la contratación o la justicia, donde las decisiones pueden tener un gran impacto en la vida de las personas.

  • Privacidad y Vigilancia

    • Con el avance de la IA, se están utilizando tecnologías que pueden recopilar y analizar grandes volúmenes de datos personales, lo que plantea serias preocupaciones sobre la privacidad. Muchas aplicaciones de IA, como el reconocimiento facial y el análisis de comportamiento, pueden seguir y monitorear a las personas en tiempo real. Esto puede llevar a una vigilancia masiva, donde las acciones y movimientos de individuos son constantemente rastreados y analizados. Uno de los principales problemas es que muchas veces las personas no son conscientes de cuántos datos están compartiendo ni de cómo se utilizan. Esto se complica por la falta de regulaciones claras y estándares éticos en el uso de estas tecnologías. Las empresas y gobiernos pueden utilizar la IA para tomar decisiones que afectan la vida de las personas, a menudo sin su consentimiento explícito.

  • Impacto en el empleo y la sociedad

    • El impacto de la inteligencia artificial en el empleo y la sociedad es un tema de gran importancia y con múltiples facetas. La IA tiene el potencial de transformar el mercado laboral de diversas maneras. Por un lado, puede automatizar tareas repetitivas y rutinarias, lo que podría llevar a la desaparición de ciertos trabajos. Por ejemplo, en sectores como la manufactura, la logística y el servicio al cliente, muchos roles pueden ser asumidos por máquinas o algoritmos, generando desempleo en esos ámbitos. Sin embargo, la automatización también puede crear nuevos empleos. A medida que las empresas adoptan tecnologías de IA, surge la necesidad de profesionales capacitados en áreas como la programación, el análisis de datos y la ética en IA. Por lo tanto, si bien algunos empleos pueden desaparecer, otros se transformarán o emergerán, generando un cambio en la demanda de habilidades. La IA también tiene el potencial de mejorar la calidad de vida al optimizar procesos en sectores como la salud, la educación y el transporte. Por ejemplo, en la atención médica, la IA puede ayudar a diagnosticar enfermedades de manera más precisa y a personalizar tratamientos. En la educación, puede facilitar el aprendizaje personalizado, adaptándose a las necesidades de cada estudiante. Sin embargo, también hay preocupaciones sobre la desigualdad social. Si el acceso a la tecnología y la educación en habilidades relacionadas con la IA no está disponible para todos, puede aumentar la brecha entre quienes pueden beneficiarse de estas tecnologías y quienes quedan atrás. Además, la dependencia excesiva de la IA podría llevar a una deshumanización en ciertos aspectos de la vida diaria, donde las interacciones humanas se ven reemplazadas por algoritmos.
